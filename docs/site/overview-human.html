mmr Retrieval Docs Overview

This local docs site is the source of truth for the search-to-read retrieval feature. It describes the finished mmr retrieve <query> workflow before code is written, so implementation can work backwards from the product contract.

The feature turns a remembered clue into a bounded packet of prior-session context. It keeps mmr find as exact literal search and adds mmr retrieve for the higher-level workflow.
