mmr retrieve Search-to-Read Pipeline

mmr retrieve <query> gives users and coding agents a single command for the workflow they were previously doing by hand: search for a phrase, decide which sessions matter, read the selected sessions, and trim the output so it is useful as context.

The command returns ranked session packets with exact match citations and bounded message windows. It deliberately stays lexical: no embeddings, no model summary, and no automatic remote fan-out. Use --all-projects and --all-sources to explicitly broaden local retrieval.

For the full product contract, read specs/retrieval.md.
